# 卡牌

> 戰鬥牌組的全部參數：類別、解鎖來源、軍費、戰場數值、單位品質、成長、用後去向、解散回收。
> **每張卡是一個有身世的部隊，不是一個數值樣板**：攻擊力與血量由卡表固定，但命中率／閃避率／攻速在取得時抽定，並隨使用成長。
> 被誰餵：區域解鎖（唯一來源）、科技（時代階級門檻）、戰鬥獎勵卡、時代演化、[[戰鬥]]（經驗與勳章）、[[營運]]（軍事區老兵、兵營勳章）。餵給誰：[[戰鬥]]、人口（解散人數型回收）、錢（刪牌／解卡費／重骰）。

## 解鎖規則（唯一來源）

- **區域開卡池**：民生區→人數型部隊卡；軍事·兵營→機械型部隊卡；軍事·兵工→工事卡＋工兵團（工事的攻防一體）；文化區→技能卡（[[營運]]）。
- **科技開階級**：每張卡有時代階級；科技達門檻（＝時代序×10；[[國策]]「文字與曆法」降為 8×時代序、「政教合一」再 +2×時代序）才能買當代階。卡隨時代**就地演化升級**，不必刪舊買新。
- **解鎖任何卡都要付錢**（解卡費＝10×時代係數；[[國策]]節點「專利制度」使其減半）。**解鎖的瞬間抽定這支部隊的品質三項**（見下方「單位品質」）——同樣是弓箭團，抽到的部隊不一樣。
- **國策不鎖卡、反而給卡**：[[國策]]樹無互斥，任何戰鬥卡都不會被封；四個戰鬥相關的節點各解鎖一張**國策限定卡**（見下表「國策限定」列）——節點完成該卡入池，照付解卡費。兩條特殊規則：
  - **技能類限定卡（勸降廣播、軌道打擊）用後即從牌組永久銷毀**，本局不再取得——一個節點換一次強效果。
  - **部隊類限定卡（聖戰士團、私掠傭兵團）與一般部隊卡同規則**：進棄牌區、可解散（無回收）、隨時代就地演化、品質照抽（見下方演化表與品質表）。
  - 「戒嚴」是秘密警察授予的主動型 [[Legacy]]（閃避一場內亂），不是卡。

## 單位品質（取得時抽定）

**只有部隊卡有品質**（工事卡與技能卡沒有：工事不出手、技能是全域效果）。解鎖／取得該卡的瞬間，以 `cards` 亂數軌在下表區間內抽定三項，**此後跟著這張卡一輩子**：

| 品質 | 是什麼 | 怎麼用 |
|---|---|---|
| **命中率** | 出手打不打得中 | 遠程單位天生低於 100%（[[戰鬥]]）；近戰穩、砲兵飄 |
| **閃避率** | 被打時閃不閃得掉 | 沒有盾值，閃避是工事以外唯一的減傷 |
| **攻速** | 一個回合窗口內出手幾次 | 決定誰先出手；先出手的活下來 |

### 品質分布表（v1 基準區間，取得時抽定）

| 部隊卡 | 命中率 | 閃避率 | 攻速（每回合出手次數） |
|---|---|---|---|
| 步兵團 | 85–95% | 5–15% | 0.9–1.2 |
| 弓箭團 | 60–80% | 10–25% | 0.8–1.1 |
| 騎兵團 | 80–90% | 15–30% | 1.0–1.4 |
| 工兵團 | —（不主動攻擊） | 5–15% | —（不出手） |
| 菁英特種部隊 | 90–98% | 20–35% | 1.1–1.5 |
| 火砲 | 55–75% | 0–10% | 0.6–0.9 |
| 轟炸機 | 70–85% | 5–15% | 0.5–0.8 |
| 聖戰士團（國策限定） | 85–95% | 0–10% | 1.2–1.6 |
| 私掠傭兵團（國策限定） | 80–90% | 10–25% | 1.0–1.3 |

- **抽到什麼就是什麼**：一支命中與攻速都抽在區間頂的弓箭團是本局的寶；抽爛的那支，要嘛認了，要嘛付錢重骰（見下方「卡牌經濟」）。這是**解散**成為真正營運決策的原因。
- **區間是設計旋鈕**：好的抽選要值得高興、爛的抽選要刺但不到不能用。

## 成長（經驗與勳章）

**練什麼強什麼**——單位的數值就是它的履歷：

| 品質 | 靠什麼累積經驗 |
|---|---|
| 命中率 | **出手攻擊**（打得越多越準） |
| 閃避率 | **被攻擊且活下來**（挨過打才學會閃） |
| 攻速 | **每存活一個回合**（純資歷） |

- **勳章**：某一項的經驗填滿，該單位即在**那個 tick 當場獲得勳章**（[[戰鬥]]：畫面看得到、不暫停），該項 +1 階，**效果自下一回合生效**。
- **成長階 v1 基準**：命中率 +3%／階、閃避率 +2%／階、攻速 +0.1／階。命中率封頂 100%、閃避率封頂 50%；**攻速無封頂**。
- **勳章的兩個來源**：
  1. **戰鬥**：經驗填滿自動獲得，加在**填滿的那一項**，不需玩家選擇。
  2. **軍事·兵營**：建築線產出勳章，玩家在**營運相位指派給任一張卡**（[[營運]]）——這是唯一由玩家挑對象的勳章。玩家挑的是**卡**，成長軸由該卡所屬列固定決定：**近戰列＝攻速、遠程列／空域（轟炸機）＝命中率、工兵團＝閃避率**（工兵團不出手，勳章只墊它唯一的成長軸）。單卡可獲勳章數**無上限**——攻速無封頂，一支近戰主力可持續疊攻速；其增幅靠兵營產出速率與成長階校準，不是結構性封頂。
- **軍事區「老兵」被動**：我方單位**起始 +1 成長階**（[[營運]]），成長軸與兵營勳章同規則（近戰＝攻速、遠程／空域＝命中率、工兵團＝閃避率）。這是純戰力、買不到，也是軍事區的唯一基礎被動；作為被動它是**常駐底線**——單位陣亡歸零後回場仍享此 +1，直到失去軍事區為止。
- **[[對手文明]]「文化國」心戰**：你的下場戰我方單位**命中率降低**。

### 陣亡與重骰

- **單位陣亡 → 該卡的已獲成長全部歸零**（含戰鬥與兵營指派的所有勳章），卡本身回到牌組（部隊還在，死的是這批老兵）。下次出場是一批新兵，從抽定值重新練起；若仍有軍事區，回場即享老兵的 +1 底線（見上）。
- **陣亡不重抽品質三項**：抽定值是這支部隊的**訓練傳統**（怎麼徵、怎麼練、怎麼打），老兵死了，傳統還在。
- 這條是**刻意的**，不是漏洞：若陣亡會重抽，故意送死就是全遊戲最便宜的重骰（送死只花 2×時代係數 軍費，還幫你擋幾下；解散再解鎖要 10×時代係數）。想重骰就付錢重骰。
- **就地演化全部保留**：抽定的品質三項與已獲勳章，跨時代演化時**一律帶著走**（[[卡牌#部隊卡的時代演化總表]]）——所以一支古典時代抽到的黃金弓箭團，可以一路帶到資訊時代；一支抽爛的，也會跟你一整局。

## 卡牌經濟

| 規則 | 內容 |
|---|---|
| 牌組大小 | 無上限、下限 5 |
| 每場出牌 | 每張卡每場只出一次（出過即進棄牌區）；**沒有手牌、沒有抽牌**，未出的卡任何回合都能出，唯一限制是軍費（[[戰鬥]]） |
| 刪牌 | 扣 8×時代係數 錢；刪人數型反而 +2 人口（窮局變現）。連同它的品質與勳章一起沒了 |
| 撤軍 | 戰鬥中隨時可退，付 10×時代係數 錢；永久 +2 人口 |
| 解散回收 | 人數型 +2 人口（墊高稅基與 BP）；機械型無回收（沉沒成本） |
| **重骰** | 解散 → 再付解卡費（10×時代係數）重新解鎖 → **重抽品質三項**。抽爛的部隊要換血，只有這條路，而且要付錢 |
| 時代演化 | 卡就地升級當代兵種（換皮換數值），不刪舊買新；品質與勳章全帶著走 |
| 戰鬥獎勵卡 | 戰鬥首見卡，勝負皆可選是否納入牌組；納入時抽定品質 |

## 兩型部隊的取捨（設計意圖）

- **人數型**：便宜、弱、退場回人口——窮局可變現的資產，餵稅基與 BP。抽爛了重骰也便宜（解散還倒賺 +2 人口）。
- **機械型**：貴、強、退場血本無歸——壓縮戰鬥回合數（少燒軍費），代價是無彈性。抽爛了重骰很痛（解散無回收），抽好了就是本局的核心資產。
- **品質是第三個軸**：兵營線的勳章可以指派、軍事區的老兵墊高起始階——軍事路線買的不是數量，是**質量**。
- 戰鬥是「輸出競速 vs 軍費燃燒」：機械型省時間、人數型省單張成本、工事卡買時間、技能卡是一次性槓桿。一副好牌通常四者都要——而且你會逐漸捨不得那幾支練出來的老兵。
