# 卡牌

> 戰鬥牌組的全部參數：類別、解鎖來源、軍費、戰場數值、用後去向、解散回收。
> 被誰餵：區域解鎖（唯一來源）、科技（時代階級門檻）、戰鬥獎勵卡、時代演化。餵給誰：[[戰鬥]]、人口（解散人數型回收）、錢（刪牌／解卡費）。

## 解鎖規則（唯一來源）

- **區域開卡池**：民生區→人數型部隊卡；軍事·兵營→機械型部隊卡；軍事·兵工→工事卡＋工兵團（工事的攻防一體）；文化區→技能卡（[[營運]]）。
- **科技開階級**：每張卡有時代階級；科技達門檻（＝時代序×10；[[國策]]「文字與曆法」降為 8×時代序、「政教合一」再 +2×時代序）才能買當代階。卡隨時代**就地演化升級**，不必刪舊買新。
- **解鎖任何卡都要付錢**（解卡費＝10×時代係數；[[國策]]節點「專利制度」使其減半）。
- **國策不鎖卡、反而給卡**：[[國策]]樹無互斥，任何戰鬥卡都不會被封；四個戰鬥相關的節點各解鎖一張**國策限定卡**（見下表「國策限定」列）——節點完成該卡入池，照付解卡費。兩條特殊規則：
  - **技能類限定卡（勸降廣播、軌道打擊）用後即從牌組永久銷毀**，本局不再取得——一個節點換一次強效果。
  - **部隊類限定卡（聖戰士團、私掠傭兵團）與一般部隊卡同規則**：進棄牌區、可解散（無回收）、隨時代就地演化（見下方演化表）。
  - 「戒嚴」是秘密警察授予的主動型 [[Legacy]]（閃避一場內亂），不是卡。

## 卡牌經濟

| 規則 | 內容 |
|---|---|
| 牌組大小 | 無上限、下限 5 |
| 刪牌 | 扣 8×時代係數 錢；刪人數型反而 +2 人口（窮局變現） |
| 撤軍 | 戰鬥中隨時可退，付 10×時代係數 錢；永久 +2 人口 |
| 解散回收 | 人數型 +2 人口（墊高稅基與 BP）；機械型無回收（沉沒成本） |
| 時代演化 | 卡就地升級當代兵種（換皮換數值），不刪舊買新；各卡的形態名見「部隊卡的時代演化總表」 |
| 戰鬥獎勵卡 | 戰鬥首見卡，勝負皆可選是否納入牌組 |

## 兩型部隊的取捨（設計意圖）

- **人數型**：便宜、弱、退場回人口——窮局可變現的資產，餵稅基與 BP。
- **機械型**：貴、強、退場血本無歸——壓縮戰鬥回合數（少燒軍費），代價是無彈性。
- 戰鬥是「輸出競速 vs 軍費燃燒」：機械型省時間、人數型省單張成本、工事卡買時間、技能卡是一次性槓桿。一副好牌通常四者都要。
