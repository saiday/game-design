# Legacy（文明精神）

> 條件觸發的永久加成＋一次署名事件。有些會隨時代衰退。Legacy 是「你這個文明是誰」的累積敘事，也是四條軸的長期放大器。
> 被誰餵：[[國策]]完成、文化領先（對在世[[對手文明]]）、[[民主]]、併吞。餵給誰：四條軸的永久加成、一張特殊[[卡牌]]。

| Legacy    | 觸發條件                                                                     | 加成                            |
| --------- | ------------------------------------------------------------------------ | ----------------------------- |
| **宗教教條**  | 國策「建立國教」完成                                                               | 文化／科技小幅永久＋；幸福大幅＋但隨時代衰退        |
| **理性精神**  | 國策「推行世俗」完成                                                               | 科技永久＋、文化小幅永久＋                 |
| **批判精神**  | 某時代文化領先所有在世對手                                                            | 科技永久＋                         |
| **搖滾精神**  | 另一個時代窗口再次文化領先所有在世對手                                                      | 人口／幸福小幅永久＋；授予技能卡「爛仗時候才宣揚愛與和平」 |
| **民主精神**  | 玩家**主動**進入民主                                                             | 人口／文化／幸福永久＋                   |
| **文化大熔爐** | 已併吞**或繼承** ≥1 個文明後，維持 10 回合（跨時代被動計數、不鎖 BP）——繼承＝退場文明因你影響力最高而併入你（[[對手文明]]） | 人口／文化／科技永久＋                   |
| **戒嚴** | 國策「秘密警察」完成 | **主動型（一次性）**：玩家可主動宣告戒嚴，無條件閃避一場內亂戰（[[內亂與失敗]]）；用掉即耗 |

- 戒嚴是目前唯一的**主動型** Legacy（其餘皆被動永久加成）：一次性消耗品，用在哪一場內亂由玩家決定。
