# 戰鬥

> 第二根支柱：只在戰鬥使用的一副牌，在單一戰場上出牌。軍費直接從錢扣（可到負）——**錢＝生命**的落點就在這裡：贏不難，難的是**用最小軍費贏**。
> 被誰餵：[[地圖與機會|地圖節點]]、[[卡牌|牌組]]、錢（軍費）、[[對手文明]]（文明戰爭）、[[內亂與失敗]]（內亂權重）、[[國策]]（偵查分支：開戰前情報）。餵給誰：錢（賠償／損失）、牌組（首見卡）、戰功歸屬（[[世界大戰]]）、失敗鏈、[[民主]]（為民主而流血）。

## 戰場結算法（單一戰場・自動佈陣）

- **一個戰場，不分路**。出牌的唯一限制是軍費：願意付錢，可以把所有部隊都擺上戰場。
- **自動佈陣**：單位依戰鬥屬性自動就位——**工事線**（最前）、**近戰列**、**遠程列**（後排）、**空域**（不佔地面位）；技能卡全域生效。玩家不指定位置。
- **每戰鬥回合**：雙方出牌（付軍費）→ 結束回合 → 播放戰鬥動畫自動結算：近戰列互擊（同時結算，攻擊力互扣 HP；HP≤0 移除並記錄清除者——戰功歸屬用）；遠程列自由選目標；帶攻城／空襲者可拆工事；騎兵類的機動與壕溝互動見[[卡牌]]。
- **勝負（每回合結算後檢查，雙方對稱）**：一方場上單位被清空、另一方場上仍有**陸軍部隊** → 陸軍在場的一方勝（只剩空中單位不算拿下戰場）。壓倒性兵力可以一回合終結戰鬥；工事與技能卡的阻擋則把戰鬥拖進下一回合，雙方繼續出牌。
- **核心博弈**：多擺一張牌就多燒一筆軍費——玩家博的是**最小花費取勝**，別讓戰鬥的軍費拖垮經濟。戰鬥 UI 全程顯示「本場已燒軍費 vs 預期賠償」，讓這個博弈看得見。
- **開戰前情報**：符合[[國策]]偵查分支的**當代情報資格** → 開戰前可見敵方牌池（本場會出的單位與卡）；未達標＝盲打（見[[國策]]）。
- **工事卡**：出到工事線，無視攻擊力。盾陣擋一次近戰、防空飛彈擋一次遠程／空襲（擋完即消耗）；**壕溝**存在時敵方近戰列無法接觸我方單位——可被敵方**工兵填平**（恢復通行）或被**坦克形態穿越**（[[卡牌]]）。同場上限 2、可被攻城／空襲摧毀。
- **判輸**＝回合上限到仍未分出勝負：自動撤軍，無獎勵、已燒軍費沉沒；內亂／獨立戰判輸另吃失敗鏈損失（[[內亂與失敗]]）。戰鬥的經濟壓力全部由軍費承擔，沒有其他直擊國庫的懲罰通道。
- **撤軍**：隨時可主動按，付 10×時代係數 錢；撤軍永久 +2 人口（人放回去）。
- **敵方單位級別**（數值×時代係數）：弱＝攻1/血2、中＝攻2/血4、硬＝攻3/血6。再乘難度係數 ×(1+0.20d)（d：Easy −1／Normal 0／Hard +1，最低 1；公式全文見[[對手文明]]）。
- **無盾值**：防禦靠工事卡或速殺。

- **戰功**＝該文明累計「清除的敵方單位實力總和」（實力＝攻+血）；**最後一擊**＝清掉敗營最後一個單位。
- 投誠只適用敘事說得通的小規模戰（一般地圖戰）；文明戰爭用心戰、內亂用讓步——三者不重疊。

## 場景呈現（戰場場景）

- 戰鬥是三個主場景之一（城市／地圖／戰場，總述見[[營運]]）：進戰即切到**專屬戰場畫面**，城市退出畫面。
- **每種戰鬥類型有自己的戰場背景**（對應戰鬥類型表的 7 型）：背景先讓玩家看懂「這是哪種仗」——收稅／地圖戰的野戰郊野、隱藏戰的異象之地（海盜／外星人）、內部暴動的城內街壘、為民主而流血的廣場、文明戰爭與世界大戰的大戰場；具體美術於資產關卡由人類定案。
- 自動佈陣的規則直接決定畫面結構：雙方相對，各自由前到後＝工事線 → 近戰列 → 遠程列 → 空域；技能卡全域生效不佔位。
- 「**本場已燒軍費 vs 預期賠償**」全程常駐；手牌與回合動作是場景內的固定 UI 帶；內亂型戰鬥依規則在開場標明民怨來源。
