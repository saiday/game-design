# 內亂與失敗

> 這個遊戲唯一的死法是一條鏈，不是一個數字：
> **欠錢 → 利息加深 → 扣幸福／內亂權重上升 → 輸掉內亂／獨立戰（失區域＋人口）→ 人口 < 5 ＝ 政權崩潰。**
> 錢永遠不直接殺你（[[經濟與債務]]）；低[[幸福]]是另一個平行入口。
> 被誰餵：債務、幸福。餵給誰：[[戰鬥]]（內部暴動戰）、[[結局]]（政權崩潰）。
> 
> 一局預期只容得下 **1–2 次**內亂失敗。

## 內亂權重（每代擲一次，≤1 場民怨戰）

| 參數                  | 值                            |
| ------------------- | ---------------------------- |
| 基礎權重                | 15%                          |
| 幸福 < 60             | +20%                         |
| 負國庫（國債能力解鎖後）        | +10% × (負額／100)              |
| [[國策]]節點（權力穩固／宗教分支） | 中央集權 −5%、秘密警察 −10%、政教合一 −10% |
| 上限                  | 60%                          |
| 頻率上限                | 每代最多 1 場民怨戰（守節奏）             |
| UI                  | 內亂戰開場標明民怨來源：低幸福 or 債務過深      |

- **讓步**：內亂戰開場可付 40×時代係數 錢取消戰鬥、幸福 +5（不打就要買）；[[國策]]「祖靈崇拜」使讓步費用減半。
- **戒嚴 [[Legacy]]**（國策「秘密警察」授予）：玩家可主動宣告戒嚴，無條件閃避一場內亂戰——一次性，用掉即耗。
- 內亂戰打贏也沒有金錢獎勵——它是純維穩懲罰，讓幸福與債務兩條軸有可見的事件出口。

## 輸掉內亂／獨立戰

- **失去 1 個區域（含其上全部建築）＋人口 −20%**。
- 已解鎖的卡保留（鎖路永不鎖卡，見[[卡牌]]）。

## 政權崩潰（GAME OVER）

- **人口 < 5 ＝政權崩潰**，唯一失敗判定（[[結局]]）。
- 介面全程顯示：目前債務深度、每回合利息、內亂權重、人口距崩潰門檻。
