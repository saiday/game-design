# 結局

> **這個遊戲沒有勝利判定的門檻——能撐到第 50 代結束、文明還在，就是勝利。** 這是標題 Insignificant 的落點：不偉大也可以活下來。

## 三種收場

| 收場 | 條件 | 性質 |
|---|---|---|
| **政權崩潰** | 人口 < 5（[[內亂與失敗]]） | **唯一的失敗**。GAME OVER |
| **走到最後** | 第 50 代結束、文明還在 | 勝利。按 power 與在世[[對手文明]]排名，給對應的**敘事結語**——只給評語、不判輸 |
| **提前完全勝利** | 只剩你一個在世文明（其餘全被你併吞，或戰敗退場被繼承，見[[對手文明]]） | 立即結束，最高規格結語 |

## 排名結語（純敘事層）

終局按玩家 power 對在世對手排名，每個名次一段結語。任何名次都是勝利的一種；數值面不接任何東西。對手滅亡後在世者變少，按實際名次取對應段落。

| 名次 | 結語（v1 文案） |
|---|---|
| **提前完全勝利** | 地圖上只剩一個名字。史書會說這是征服——你知道，這只是活得比所有人都久。從今天起，「微不足道」這個詞由你來定義別人。 |
| 第 1 名 | 你不再微不足道。從夾縫裡的小部落到世界的第一強權——史官寫到這裡停了很久，因為找不到先例可以引用。 |
| 第 2 名 | 你站在世界的第二位。那個第一名，曾經連你的名字都懶得記——現在他們的國策會議上，你是第一個議題。 |
| 第 3 名 | 不是最強的，也遠不是最弱的。你的文明成了其他小國的教科書：夾縫裡不只能活，還能活得體面。 |
| 第 4 名 | 排名不好看，但排名是強權的遊戲。你的人民記得的是另一份清單：熬過的戰爭、還清的債、沒有崩塌的每一天。 |
| 第 5 名 | 大國的史書裡，你只是一個註腳。但註腳的意思是：五十個世代過去，他們仍然不得不提到你。 |
| 第 6 名 | 你活下來了——而活下來就夠了。這就是一切的答案：微不足道，但仍然在這裡。 |

## 永久性

- 不能讀檔反悔；每一局的決定都是最終的。
- 介面全程顯示距離失敗有多遠：債務深度、利息、內亂權重、人口距崩潰門檻。
